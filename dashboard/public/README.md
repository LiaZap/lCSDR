# Assets estáticos

Coloque aqui:

- **`lc-logo.png`** — logo oficial da LC Agência (PNG com fundo transparente).
  O componente `<LCLogo />` carrega automaticamente este arquivo se existir; caso contrário usa um fallback CSS que reproduz o logo com a wordmark "LC" + dois balões de fala magenta/berry.

- **`lc-logo-white.png`** (opcional) — versão pra fundos escuros.

- **`favicon.ico`** — favicon do dashboard.

Dimensões sugeridas: `400x160` @ 2x (PNG). Transparente.
